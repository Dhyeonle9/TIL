# 1회차 강의 내용 요약
1. Git
        - git 기본 커맨드
        - 초기 설정 (`config`, `init`, `remote add`)
        - working directory 내용 수정/ staging area 업로드하기 (`add`)
        - snapshot 찍기 (`commit`)
        - 원격 저장소에 업로드하기 (`push`)
2. Linux
    - 작업중인 폴더 확인 (`pwd`)
    - 작업중인 디렉토리 내의 파일/폴더 출력 (`ls`)
    - 디렉토리 이동 (`cd`)
    - 폴더 생성 (`mkdir`)
    - 파일 생성 (`touch`)
    - 파일/폴더 제거 (`rm <파일이름.확장자>/'rm -r <폴더이름>`)
3. Markdown
    - 기본 구문 학습
    - url연결 (`[문구](url)`)
    - 이미지삽입 (`![대체문구](이미지주소)`)
    - 코드블럭 ( ``` backtick 3개 사이에 코드입력)
